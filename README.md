<b>二分法（Bisection method）</b>: 是一種方程式根的近似值求法。 <br>

<h4>演算法執行步驟</h4>
若要求已知函數 f(x) = 0 的根 (x 的解)，則:
<ol>
  <li>先找出一個區間 [a, b]，使得f(a)與f(b)異號。根據勘根定理，這個區間內一定包含著方程式的根。</li>
  <li>求該區間的中點 `$m=\frac{a+b}{2}$`，並找出 f(m) 的值。$\frac{a+1}{b+2}$</li>
    <li> `$\frac{a+1}{b+2}$` ==> $\frac{a+1}{b+2}$</li>
</ol>
